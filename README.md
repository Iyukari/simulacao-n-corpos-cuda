# Implementação e Análise de Desempenho da Simulação de N-Corpos em GPU Utilizando CUDA

## Integrantes
- Bruno Alves Mendonça de Abreu Filho
- Emily Iyukari Soma Hamano
- Pedro Costa Freitas

## Disciplina
Arquitetura de Computadores — 2026/1

## Objetivo
Este projeto tem como objetivo implementar uma simulação de N-corpos utilizando CPU e GPU com CUDA, comparando desempenho, speedup e otimizações utilizando memória compartilhada.

## Tecnologias utilizadas
- Python
- CUDA
- Numba
- Google Colab
- Matplotlib

## Estrutura do repositório

```text
notebook/     -> códigos e notebooks
resultados/   -> gráficos, tabelas e imagens
slides/       -> apresentação
artigo/       -> relatório SBC
```

## Execução
O notebook pode ser executado diretamente no Google Colab com suporte a GPU.
